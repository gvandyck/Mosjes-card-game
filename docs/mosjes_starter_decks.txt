================================================================================
NOTES FOR IMPLEMENTATION
================================================================================

1. Each deck is self-contained and balanced for 2-player matches

2. Quest deck should be shared between both players (shuffle 20-30 total Quests)

3. Place cards can be included in personal decks OR in a shared Place deck

4. Starting hand: 5 cards, starting MP: varies by Mosje (see Mosje tables)

5. Players should select 2 Mosjes from their deck before shuffling

6. All three decks use different strategies but are balanced against each other

7. These are starter decks - advanced players can customize and build their 
   own decks using the full card database

8. For testing purposes, have both players use different decks to test various 
   gameplay interactions

================================================================================
QUICK REFERENCE - GAME SETUP
================================================================================

STEP 1: Each player chooses one of the three decks

STEP 2: Each player selects 2 Mosjes from their deck (place them in Mosje slots)

STEP 3: Shuffle remaining cards (without the 2 selected Mosjes)

STEP 4: Create shared Quest deck (shuffle 20-30 Quest cards from combined pool)

STEP 5: Each player draws 5 cards as starting hand

STEP 6: Set each Mosje's MP to their starting MP value (varies by Mosje)

STEP 7: All Mosjes start at Level 1

STEP 8: Player 1 begins their turn
